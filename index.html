API Nastavení

Pro používání HejChat potřebujete API klíč z OpenRouter:

1

Registrace na OpenRouter

Nejprve přejděte na web OpenRouter a zaregistrujte se pro vytvoření účtu.

2

Vytvoření API klíče

V nastavení vašeho OpenRouter účtu vytvořte nový API klíč.

3

Vložení API klíče do HejChat

Zkopírujte vytvořený API klíč a vložte jej do HejChat v sekci API.
